# B-Manager 機能・操作ガイド

## 【重要】VCC版移行に伴うデータ互換性について（必ずお読みください）
**VCC対応以前のバージョン（旧版）をご利用の方へ**
本バージョンよりデータ保存の仕様が変更されました。そのため、**旧バージョンからのデータ移行機能はありません。**

- VCC管理へ移行（本パッケージを導入）した場合、以前のバージョンで登録していたデータは**破損（参照不可）状態**となります。
- 大変お手数ですが、VCC版導入後は**すべてのアイテムを新規に再登録**していただく必要があります。
あらかじめご了承の上、ご利用ください。

## ■ 主な機能
- **アセット紐付け**: プロジェクト内のファイルと販売ページ情報を統合管理。
- **自動情報取得**: BOOTHのURLから商品名とサムネイル画像を自動取得。
- **データの安全性**: アイテム保存時にテキストデータ（JSON）を自動バックアップし、万が一のデータ破損リスクを軽減。
- **インポート検知**: 新規アセット導入時に登録画面を自動表示（ON/OFF切替可）。
- **カテゴリー分類**: タブによる表示切り替えと、スリムな矢印でのスクロール。
- **検索・ソート**: キーワード検索および名前・登録日順での並び替え。
- **一括更新**: 登録済みアイテムの情報を一括で最新の状態へ更新。

## ■ 使用手順

### 1. ウィンドウの起動
- Unityメニューの `UniMuse.lab` > `B-Manager` を選択してメインウィンドウを開きます。

### 2. アイテムを登録する
- **右クリック**: プロジェクト内のアセットを右クリックし `Register to B-Manager` を選択。
- **ドラッグ&ドロップ**: アセットを直接 B-Manager ウィンドウへドラッグ。
- **自動表示**: `Auto Open` を有効にしている場合、インポート時に自動で登録画面が開きます。

### 3. 情報の保存
- 登録画面で商品URLを入力し「情報を取得して保存」をクリックします。
- BOOTH以外のURLや手動で登録したい場合は「入力内容のみで保存」を選択します。
- **保存データ**: データはプロジェクト内の `Assets/UniMuseData` フォルダに自動保存されます。

### 4. 閲覧・管理
- リスト内のアイテムをクリックすると、該当アセットをプロジェクト内でハイライトします。
- カテゴリータブを切り替えることで、特定のジャンルのみを表示できます。
- アイテムを右クリックすると、URLを開く、再取得、削除などのメニューが表示されます。
- **削除時**: リストから削除を行うと、データファイルとバックアップ（JSON）の両方が整理されます。

### 5. 登録画面を閉じる
- ポップアップ内の「キャンセル」ボタンを押すと閉じます。
