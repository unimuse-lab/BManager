# B-Manager

**B-Manager** は、Unity プロジェクト内のアセットと BOOTH などの販売ページ情報を紐づけて管理するエディタ拡張ツールです。

---

## 【重要】VCC版移行に伴うデータ互換性について（必ずお読みください）

**VCC対応以前のバージョン（旧版）をご利用の方へ**

本バージョンよりデータ保存の仕様が変更されました。そのため、**旧バージョンからのデータ移行機能はありません。**

- VCC管理へ移行（本パッケージを導入）した場合、以前のバージョンで登録していたデータは**破損（参照不可）状態**となります。
- 大変お手数ですが、VCC版導入後は**すべてのアイテムを新規に再登録**していただく必要があります。

あらかじめご了承の上、ご利用ください。

---

## 主な機能

- **アセット紐付け**: プロジェクト内のフォルダと販売ページ情報を統合管理
- **フォルダ階層選択**: インポートフォルダに配下フォルダがある場合、紐づけるフォルダをツリーUIで選択可能
- **自動情報取得**: BOOTHのURLから商品名とサムネイル画像を自動取得
- **サムネイル複数選択**: BOOTHページ内の全サムネイル候補を一覧表示し、好みの画像を選択可能
- **データの安全性**: アイテム保存時にテキストデータ（JSON）を自動バックアップし、万が一のデータ破損リスクを軽減
- **インポート検知**: `.unitypackage` の新規インポート時に登録画面を自動表示（ON/OFF切替可）
- **カラーハイライト**: 最後に登録したフォルダをProjectウィンドウ上でカラー表示（色は自由に設定可能）
- **カテゴリー分類**: タブによる表示切り替えと、スリムな矢印でのスクロール
- **検索・ソート**: キーワード検索および名前・登録日順での並び替え
- **一括更新**: 登録済みアイテムの情報を一括で最新の状態へ更新

## 使用手順

### 1. ウィンドウの起動

Unityメニューの `UniMuse.lab` > `B-Manager` を選択してメインウィンドウを開きます。

### 2. アイテムを登録する

以下の3つの方法で登録できます。

| 方法 | 操作 |
|------|------|
| **右クリック** | Projectウィンドウのフォルダを右クリック → `Register to B-Manager` を選択 |
| **ドラッグ&ドロップ** | アセットをB-Managerウィンドウへ直接ドラッグ |
| **自動表示** | `Auto Open` を有効にしている場合、`.unitypackage` インポート時に自動で登録画面が開く |

> **注意**: Hierarchyウィンドウ上のオブジェクトは登録できません。Projectウィンドウ内の永続的なアセット（フォルダ）のみが対象です。

### 3. 登録画面の操作

#### フォルダ階層の選択
インポートしたフォルダの配下にサブフォルダが存在する場合、どのフォルダを紐づけるか選択できます。

- 登録済みのフォルダはグレー表示（✓登録済み）で選択不可
- 未登録のフォルダのみ選択可能
- サブフォルダを選択した場合でも、Projectウィンドウのカラーハイライトはルートフォルダに適用されます

#### サムネイルの設定

| 設定 | 動作 |
|------|------|
| ☑ トップサムネイルを自動取得（デフォルト） | BOOTHページのメイン画像（og:image）を自動で取得 |
| ☐ チェックなし | ページ内の全サムネイル候補を一覧表示し、好みの画像を選択可能 |

> 再取得時は、前回選択したサムネイルが引き続き存在する場合はそのURLを優先して取得します。
> サムネイルなしで登録したい場合は「入力内容のみで保存」を使用してください。

#### 情報の保存

- **「情報を取得して保存」**: BOOTH URLを入力している場合に有効。商品名・サムネイルを自動取得して保存
- **「入力内容のみで保存」**: BOOTH以外のURLや手動入力内容をそのまま保存
- **「キャンセル」**: 登録画面を閉じます（データは保存されません）

保存データは `Assets/UniMuseData/B-Manager` フォルダに自動保存されます。

### 4. 閲覧・管理

| 操作 | 内容 |
|------|------|
| アイテムをクリック | 該当アセットをProjectウィンドウでハイライト表示 |
| カテゴリータブ切り替え | 特定のジャンルのみを表示 |
| アイテムを右クリック | URLを開く / 編集 / 再取得 / 削除などのメニューを表示 |

**右クリックメニュー詳細**

| メニュー | 内容 |
|---------|------|
| URLを開く | BOOTHページをブラウザで開く |
| 編集 | 登録情報を編集画面で開き直す |
| 再取得 | BOOTHから最新情報を再取得（BOOTH URLのみ） |
| リストから削除 | BManagerデータのみ削除 |
| アセット本体も削除 | BManagerデータ + インポートフォルダを削除 |

> **削除時**: リストから削除を行うと、データファイルとバックアップ（JSON）の両方が整理されます。

### 5. カラーハイライトと設定

ツールバーの設定ボタン（⚙）から以下の設定が可能です。

| 設定 | 内容 |
|------|------|
| 最後に追加されたフォルダをハイライト | 直近の登録フォルダをProjectウィンドウ上でカラー表示（ON/OFF） |
| ハイライトカラー | 表示色を自由に変更可能 |
| ハイライトをクリア | ハイライト表示を手動で消去 |
